# Portfolio Tracker v2.0.1 — Initial Public Release

**Released:** 2026-07-13

Portfolio Tracker 2.0 is now available to the public. Version 2.0.1 is the first release to licensed users, featuring a complete rewrite of the original Excel-based application with a standalone Windows installer.

## Install

Download [PortfolioTracker-v2.0.1-Setup.exe](https://github.com/davidshuvalov/portfolio-tracker-releases/releases/download/v2.0.1/PortfolioTracker-v2.0.1-Setup.exe) and double-click to install.

- Works on Windows 10 / 11 (64-bit)
- No Python or other software required
- Choose **all-users** (`Program Files`) or **just-for-me** (`AppData`) at install time; desktop shortcut optional

To upgrade from a previous build, re-run the new `Setup.exe` — it will upgrade in place.

## Highlights

### Portfolio management

- **Portfolio Optimizer** — multi-step build workflow with strategy selection, symbol/sector concentration rules, per-strategy exposure limits, and Monte Carlo risk-of-ruin targeting
- **Inline editable contracts grid** — adjust contracts directly in the portfolio table
- **Auto-rebuild on Apply** — portfolio updates immediately when optimizer results are applied
- **Position Check** — moved to Portfolio functions for portfolio-level validation
- **Re-optimisation Due** — reliable table on the Build Portfolio page

### Strategy analysis

- **Strategy Tracker** — colour-coded status, contract-fit indicators, time-in-market (trading days), and auto-computed metrics on page load
- **Strategy Detail** — walkforward performance and inputs as top-level tabs; ELCode and folder quick-open buttons; single-contract evaluation
- **Strategy Screener** — status/portfolio action bar with readable rule columns
- **Strategy Compare** — side-by-side comparison with OOS period options
- **Eligibility Backtest** — reworked rule statistics with selectable baseline and data scope
- **Health Dashboard** and **Portfolio Suggestions**

### Risk and simulation

- **Monte Carlo** — per-strategy tab (solve for risk of ruin); portfolio-level MC with period selection (IS / OOS / all / last N years); sample equity curve output
- **Closed-trade MC** — opt-in toggle with margin stop and RTD reporting
- **Stress Scenario / Crisis Replay** — portfolio stress testing page
- **ATR Rebalance** — per-strategy tab aligned with the original spreadsheet logic

### Benchmarking and market context

- **Portfolio vs Benchmark** — alpha, beta, and correlation vs scaled buy-and-hold
- **Benchmark page** — time-period control and three Live Backtest reconstruction modes
- **Market Summary** — calendar analysis tab; strategy-vs-market moved to portfolio page with per-strategy Market Conditions tab

### Contract sizing and margins

- **Contract Sizing** — margin and ATR multiple sliders (0–3× range); override column with effective values and decimal contract charts; moved to Portfolio navigation
- **Margin Tracking** — TradeStation, Interactive Brokers, and NinjaTrader margin sources with reimport; 3-month rolling and all-time average on Total Portfolio Margin chart
- **Margin concentration** — time-weighted by in-market frequency; configurable exposure shares; tradeable contract steps with auto-micro switching
- **Broker seed data** — margins and contract mapping seeded from the original Portfolio Tracker workbook

### MultiWalk integration

- **Setup file import** — automatic path rewriting when receiving or moving MultiWalk project folders
- **ELCode @ Params** — open strategy code at the current parameter window
- **Walkforward Files scan** — finds `ELCode.txt` in the Walkforward Files subfolder
- **WF Dashboard** — split into Performance and Inputs sub-tabs with copy-ready input blocks

### Exports and branding

- **PDF performance summaries** — per-strategy and full-portfolio exports
- **Official logo design** — candlestick icon and gold wordmark applied across the app

### User experience

- **Expand/Collapse** toggle on every table across the app
- **Column filters** — reusable filters on strategy tables; filters and column views persist across page navigation
- **Configurable status options** — custom status labels with colour coding in Settings
- **Crisp table rendering** — fixed-height scroll boxes with sticky headers; HTML rendering for expanded tables
- **Pagination** on portfolio metrics table to fix scroll blur

### Licensing and packaging

- Standalone Windows installer — no Python runtime required
- Offline license master secret rotated for the final release
- MultiWalk DLL released after each license check (no file lock)

## Bug fixes in this release

- Fixed Portfolio at a Glance to properly apply configured status options

## Upgrading from the spreadsheet

Portfolio Tracker 2.0 reads the same MultiWalk walkforward folder structure as the original workbook. Point the app at your existing MultiWalk project root on first launch. Broker margins and contract mappings are pre-seeded from the spreadsheet defaults.
